# Glossary

- Native IQ: Product name for Slack-focused insights and automation by DeepNative.
- Insight: Normalized object capturing decision, risk, blocker, trend, or summary with sources.
- Task Miner: Component that extracts structured tasks from Slack threads.
- MCP: Model Context Protocol; hub and servers expose tools to the LLM.
- ACL: Access Control List restricting visibility by channel/user/tenant.
- RLS: Row-Level Security in Postgres enforcing tenant isolation.
- Home Tab: Slack app surface showing personalized blocks.
